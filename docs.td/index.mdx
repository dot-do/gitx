A complete Git implementation designed for edge computing with Cloudflare's infrastructure. Build git-powered applications with full packfile support, smart HTTP protocol, MCP integration for AI workflows, and intelligent tiered storage.

## Installation

```bash
npm install gitx.do
```

## Core Features

### Pack Files

Full Git packfile v2/v3 support with delta compression. Parse, create, and verify pack indexes with efficient binary operations.

```typescript
import { parsePackIndex, createPackIndex, lookupObject } from 'gitx.do/pack'

const index = parsePackIndex(indexData)
const entry = lookupObject(index, 'abc123...')
```

### Wire Protocol

Smart HTTP protocol implementation for fetch/push operations compatible with standard Git clients.

```typescript
import { handleInfoRefs, handleUploadPack, handleReceivePack } from 'gitx.do/wire/smart-http'

const response = await handleInfoRefs(request, repository, {
  sideBand64k: true,
  ofsDelta: true
})
```

### MCP Tools

Model Context Protocol integration enables AI-assisted git operations through a standardized tool interface.

```typescript
import { gitTools, invokeTool } from 'gitx.do/mcp/tools'

const result = await invokeTool('git_status', { path: '/repo', short: true })
```

### Tiered Storage

Intelligent hot/warm/cold storage tiers with automatic promotion:

- **Hot**: Durable Object SQLite for low-latency access
- **Warm**: R2 object storage for packed objects
- **Cold**: Analytics/Parquet for archival

```typescript
import { TieredReader } from 'gitx.do/tiered/read-path'

const reader = new TieredReader(hotBackend, warmBackend, coldBackend, {
  promotionPolicy: 'aggressive'
})
```
